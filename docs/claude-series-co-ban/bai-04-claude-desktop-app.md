# Bài 4: Claude Desktop App - AI trên máy tính

> Series: Làm Quen Với Claude AI | Cộng đồng Claude AI VN

## Claude Desktop là gì?

Claude Desktop là ứng dụng native cho macOS và Windows, cho phép bạn:
- Chat với Claude mà không cần mở browser
- Tích hợp với hệ thống qua MCP (Model Context Protocol)
- Truy cập nhanh bằng keyboard shortcut

## Download & Cài đặt

### macOS
1. Truy cập [claude.ai](https://claude.ai)
2. Click **"Download for macOS"**
3. Mở file `.dmg`
4. Kéo Claude vào Applications
5. Mở app, đăng nhập

### Windows
1. Truy cập [claude.ai](https://claude.ai)
2. Click **"Download for Windows"** (hoặc Windows arm64)
3. Chạy installer `.exe`
4. Follow hướng dẫn
5. Mở app, đăng nhập

## Tính năng chính

### 1. Native Experience
- Nhanh hơn web
- Chạy như app riêng
- Không cần browser

### 2. Keyboard Shortcut
```
macOS:  Cmd + Space (có thể customize)
Windows: Ctrl + Space
```
→ Mở Claude nhanh từ bất kỳ đâu

### 3. MCP Integration
- Kết nối với local files
- Tích hợp với tools khác
- Xem chi tiết ở Bài 6

### 4. System Tray
- Chạy nền
- Quick access
- Notifications

## Desktop vs Web

| Tiêu chí | Desktop | Web |
|----------|---------|-----|
| Speed | Nhanh hơn | Bình thường |
| Offline | Không | Không |
| MCP support | ✅ Có | ❌ Không |
| Multi-account | ❌ | ✅ Tabs |
| Quick access | ✅ Shortcut | ❌ |

## Cấu hình cơ bản

### Thay đổi keyboard shortcut (macOS)
1. Mở Claude Desktop
2. Menu: Claude → Preferences
3. Tab: Shortcuts
4. Đổi shortcut theo ý muốn

### Appearance
- Light/Dark mode
- Follow system

## Khi nào nên dùng Desktop?

**Nên dùng Desktop:**
- Cần truy cập nhanh, thường xuyên
- Muốn dùng MCP
- Thích native app experience

**Nên dùng Web:**
- Dùng nhiều tài khoản
- Máy yếu, tiết kiệm RAM
- Dùng trên máy công cộng

## Tips

1. **Pin to Dock/Taskbar** - Truy cập nhanh
2. **Set startup** - Tự chạy khi bật máy
3. **Learn shortcut** - Tiết kiệm thời gian
