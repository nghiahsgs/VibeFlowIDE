## Tổng quan 3 Models

Claude có 3 model chính, mỗi model phục vụ mục đích khác nhau:

```
Opus 4.5   → Thông minh nhất, đắt nhất
Sonnet 4.5 → Cân bằng giữa chất lượng và chi phí
Haiku 4.5  → Nhanh nhất, rẻ nhất
```

## So sánh chi tiết

### Claude Opus 4.5
- **Mô tả:** Model thông minh nhất cho agents và coding
- **Điểm mạnh:** Complex reasoning, coding tasks, agentic workflows
- **Khi nào dùng:** Tasks khó, cần chất lượng cao nhất
- **API Pricing:** Input $5/MTok | Output $25/MTok

### Claude Sonnet 4.5
- **Mô tả:** Cân bằng tối ưu giữa intelligence, cost, speed
- **Điểm mạnh:** Đa năng, phù hợp hầu hết use cases
- **Khi nào dùng:** Daily tasks, coding, writing, analysis
- **API Pricing:** Input $3-6/MTok | Output $15-22.5/MTok

### Claude Haiku 4.5
- **Mô tả:** Nhanh nhất, tiết kiệm chi phí nhất
- **Điểm mạnh:** Speed, cost-efficiency, simple tasks
- **Khi nào dùng:** Chatbots, classification, quick responses
- **API Pricing:** Input $1/MTok | Output $5/MTok
